# Taxonomía SALI

La taxonomía SALI es el sistema de clasificación de BrainLex. Permite etiquetar contactos, expedientes y documentos con categorías estandarizadas para facilitar la búsqueda y la organización.

## Las cinco categorías

### Identidad

Etiquetas que describen **qué es** el contacto: persona física, empresa, fundación, administración pública, etc.

### Departamento

El área de trabajo que gestiona la relación: Fiscal, Laboral, Extranjería, Mercantil, Civil, etc.

### Servicio

El tipo de servicio contratado: IRPF, Nóminas, Permisos de Residencia, Constitución de Sociedades, etc.

### Estado

Etiquetas de estado de proceso: Facturado, Pendiente, En curso, Blueprint, etc. Son etiquetas del sistema que no se pueden modificar.

### Inteligencia

Etiquetas personalizadas para clasificación adicional: VIP, Urgente, Recurrente, etc.

## Cómo asignar etiquetas

Desde la ficha del contacto, pestaña **Ecosistema**:

1. Busca la sección de taxonomía SALI
2. Arrastra una etiqueta desde el **Constructor** (panel lateral) a la zona de asignación del contacto
3. La etiqueta queda vinculada al contacto

> También puedes hacer clic directamente en una etiqueta para asignarla sin arrastrar.

## El Constructor de etiquetas

El Constructor es el panel donde se gestionan las etiquetas disponibles. Solo los usuarios con permisos de administrador pueden **crear etiquetas nuevas** (etiquetas raíz). El resto de usuarios puede asignar las existentes.

### Crear una etiqueta nueva

1. Ve a **Administración → Taxonomía**
2. Selecciona la categoría (Departamento, Servicio, etc.)
3. Escribe el nombre de la nueva etiqueta
4. Confirma la creación

### Reordenar etiquetas

Las etiquetas se pueden reordenar dentro de su categoría arrastrándolas con el icono de agarre (las tres líneas horizontales a la izquierda de cada etiqueta).

## Impacto en la Bóveda

Las etiquetas de tipo **Departamento**, **Servicio** y **Estado** influyen directamente en la estructura de carpetas Blueprint de la Bóveda Documental. Cuando asignas una etiqueta de servicio a un contacto, se genera automáticamente la carpeta correspondiente en su bóveda.
